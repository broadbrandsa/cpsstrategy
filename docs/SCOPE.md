# Scope

## Pages Included

- **Home / Landing Page** — Hero, value proposition, key sections overview
- Additional pages TBD based on strategy content requirements

## Sections Included

- Hero / Above the fold
- Strategic overview
- Capabilities / Services
- Approach / Methodology
- Results / Proof points
- Team / About
- Contact / Next steps
- Footer

## What Is Excluded

- Backend API or database integration
- User authentication or gated content
- E-commerce or payment processing
- Blog or CMS functionality
- Multi-language / i18n support
- Email sending infrastructure (forms will be tracking-only or external)

## Launch Milestones

1. **Scaffold Complete** — Project structure, tooling, and docs in place
2. **Content Draft** — All section content authored in `src/content`
3. **Design Implementation** — Sections built with shadcn + Tailwind
4. **Review & QA** — Internal review, responsiveness check, accessibility pass
5. **Deploy to Vercel** — Production deployment with custom domain (if applicable)
