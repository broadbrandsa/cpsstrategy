# Project Overview

## What This Is

CPS Strategy is a standalone proposal/strategy website built to present CPS's strategic direction, capabilities, and value proposition to target audiences. It serves as both an internal alignment tool and an external-facing presentation layer.

## Target Audience

- Prospective clients and partners evaluating CPS
- Internal stakeholders reviewing strategic direction
- Decision-makers requiring a clear, visual summary of CPS offerings

## Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Components:** shadcn UI primitives
- **Package Manager:** pnpm
- **Deployment:** Vercel

## Deployment Environment

- **Hosting:** Vercel (production)
- **CI/CD:** Vercel auto-deploy from GitHub `main` branch
